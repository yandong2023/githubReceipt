# GitHub Receipt Generator 🧾

A modern, elegant web application that generates stylish receipts from your GitHub profile data. Built with pure HTML and TailwindCSS, this project creates beautiful, shareable receipts showcasing your GitHub contributions and activity.

## ✨ Features

- **Instant Receipt Generation**: Enter any GitHub username and get a beautifully formatted receipt
- **Detailed GitHub Stats**:
  - Repository count
  - Stars earned
  - Forks received
  - Followers & Following
  - Top programming languages
  - Most active day
  - 30-day commit count
  - Contribution score
- **Professional Design**:
  - Clean, receipt-like layout
  - Custom barcode generation
  - Unique coupon codes
  - Card information styling
  - Dashed borders and decorative elements
- **Share & Download**:
  - Download receipt as image
  - Share directly with others
  - URL parameter support for shared receipts

## 🚀 Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Enter a GitHub username
4. Click "Generate Receipt"

## 🛠 Technologies Used

- **HTML5**: Core structure and semantic markup
- **TailwindCSS**: Styling and responsive design (via CDN)
- **JavaScript**: Dynamic content and interactions
- **GitHub API**: Fetching user data and statistics
- **External Libraries**:
  - html2canvas: Receipt image generation
  - JsBarcode: Barcode creation
  - Google Fonts (Courier Prime)
